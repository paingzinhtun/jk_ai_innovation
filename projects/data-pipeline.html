Project Overview

As part of JK AI Innovation, I built this pipeline to automate the collection and cleaning of spiritual teachings. This ensures that the data is ready for future AI model training.

Technical Process

  • Extract: Pulling raw quote data from CSV/Text sources.
  • Transform: Using the Pandas library to clean text and add timestamps.
  • Load: Saving structured data for analysis.
